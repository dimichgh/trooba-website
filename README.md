# Source code for trooba.com

This repo contains the source code for trooba.com.

## Running trooba.com Locally

``` bash
git clone https://github.com/trooba/trooba-website.git
cd trooba-website
npm install
npm start
```

## Serving a static build

```bash
npm run serve-static
```

## Publishing

```bash
npm run publish
```
